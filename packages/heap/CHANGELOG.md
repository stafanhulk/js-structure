## [1.0.0] - 2025-12-29

### Added

#### 核心功能

- 实现完整的堆数据结构，支持泛型类型
- 提供 `MIN_HEAP` 和 `MAX_HEAP` 两种预设比较器
- 支持自定义比较器函数

#### 基本操作

- `push(value)` - 向堆中添加元素，自动维护堆属性
- `pop()` - 移除并返回堆顶元素
- `top()` - 查看堆顶元素但不移除
- `isEmpty()` - 检查堆是否为空
- `size()` - 获取堆中元素数量

#### 工具方法

- `clear()` - 清空堆中所有元素
- `clone()` - 创建堆的深拷贝
- `toArray()` - 将堆转换为数组
- `balance()` - 对无序数组进行堆化操作
- `isValid()` - 验证堆属性是否正确

#### 高级特性

- 实现 `Symbol.iterator` 接口，支持 `for...of` 循环
- 支持展开运算符 `[...heap]`
- 支持链式调用（push、clear、balance 方法）
- 完整的 TypeScript 类型支持

#### 性能特点

- 插入操作时间复杂度：O(log n)
- 删除操作时间复杂度：O(log n)
- 查看堆顶时间复杂度：O(1)
- 堆化操作时间复杂度：O(n)

#### 构建和发布

- 同时支持 CommonJS 和 ESM 模块格式
- 提供完整的 TypeScript 类型声明文件
- 包含完整的单元测试（100% 覆盖率）
- 支持在 Node.js 和浏览器环境中使用

### 应用场景

- 优先队列实现
- Top K 问题求解
- 任务调度系统
- 中位数维护
- 堆排序算法
